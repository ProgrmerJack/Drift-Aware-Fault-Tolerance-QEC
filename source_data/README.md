This directory contains figure source data files per Nature policy requirements.

```

│   ├── fig2a_t1_timeseries.csv
```

## Excel File Structure (SourceData.xlsx)

The Excel file contains one sheet per figure panel:

| Sheet Name | Description | Key Columns |
|------------|-------------|-------------|
| Fig1a | Concept schematic data | component, error_rate, description |
| Fig1b | QPU budget allocation | method, shots, time_seconds |
| Fig1c | Pipeline comparison | method, error_rate, ci_lower, ci_upper |
| Fig2a | T₁ time series | timestamp, qubit_id, t1_us, backend |
| Fig2b | Ranking stability | session_pair, kendall_tau, hours_elapsed |
| Fig2c | Drift heatmap | qubit_id, metric, z_score |
| Fig2d | Calibration gap | time_since_cal_h, optimal_qubit_match |
| Fig3a | Syndrome sequences | shot_id, round, syndrome_bits |
| Fig3b | Burst distribution | burst_length, count, expected_iid |
| Fig3c | Spatial correlation | qubit_i, qubit_j, correlation |
| Fig3d | Temporal statistics | metric, value, ci_lower, ci_upper |
| Fig4a | Error comparison | session_id, baseline_error, adaptive_error |
| Fig4b | Paired differences | session_id, diff, backend |
| Fig4c | CI visualization | method, estimate, ci_lower, ci_upper |
| Fig4d | Effect sizes | comparison, cohens_d, ci_lower, ci_upper |
| Fig5a | Backend generalization | backend, effect_size, p_value |
| Fig5b | Distance scaling | code_distance, baseline, adaptive |
| Fig5c | Resource efficiency | probe_shots, improvement, overhead |

## Data Format

All numeric data are stored with appropriate precision:
- Error rates: 6 decimal places
- Time values: 2 decimal places
- Statistical metrics: 4 decimal places
- P-values: Scientific notation when < 0.001

## Generation

Source data files are generated by:
```bash
python analysis/generate_source_data.py
```

## Nature Policy Compliance

Per Nature Communications policy:
- All data plotted in figures must have corresponding source data
- Excel file must be uploadable via the submission system
- Individual CSV files provide machine-readable access
- Missing values are indicated with "NA" (not blank cells)
